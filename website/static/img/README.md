# MACHUPS Brand Assets

This directory contains brand assets for the documentation site.

## Required Images

### Logo
- **logo.svg** - MACHUPS logo (will be created)
- **favicon.ico** - Favicon (will be created)

### Social Card
- **machups-social-card.png** - Open Graph image (1200x630px)
  - Should include MACHUPS branding
  - Include tagline: "AI-Powered Brand Generation in Under 3 Minutes"
  - Show example generated logos
  - Monad branding/badge

## Temporary Placeholders

Until proper assets are created, the site references these files. They should be created with MACHUPS branding before production deployment.

### Logo Guidelines
- Primary color: #0066FF
- Font: Inter or similar modern sans-serif
- Minimalist, tech-forward design
- Works well at small sizes

### Social Card Guidelines
- Dimensions: 1200x630px
- Format: PNG
- Include:
  - MACHUPS logo
  - Tagline
  - Example output (logo samples)
  - "Built on Monad" badge
  - Clean, modern design

## Creating Assets

1. Use Figma or similar design tool
2. Export logo as SVG (vector)
3. Export social card as PNG (1200x630)
4. Convert logo SVG to favicon.ico (16x16, 32x32, 48x48)
5. Place files in this directory
6. Commit and deploy
