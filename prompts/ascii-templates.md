## Usage Guidelines

### Component Breakdown

Each template includes:
- **Header/Navigation**: Logo, main nav, CTA buttons
- **Hero Section**: Primary message and action
- **Content Area**: Main interactive elements
- **Footer**: Links, social, legal

### CTA Placement Best Practices

1. **Above the Fold**: Primary CTA in hero section
2. **End of Sections**: Secondary CTAs after value propositions
3. **Footer**: Tertiary CTAs for re-engagement
4. **Sticky Elements**: Optional for high-priority actions

### Accessibility Notes

- **Keyboard Navigation**: All interactive elements must be keyboard accessible
- **Screen Readers**: Use semantic HTML and ARIA labels
- **Focus States**: Clear visual indicators for keyboard navigation
- **Touch Targets**: Minimum 48x48px on mobile
- **Color Contrast**: WCAG AA minimum (AAA preferred)

### Responsive Breakpoints

- **Mobile**: 320px - 768px (40 char width)
- **Tablet**: 768px - 1024px (60 char width)
- **Desktop**: 1024px+ (80 char width)

## Integration with Brand Generator

These templates are suggested based on:
- **Business Type**: Detected from user input
- **Target Audience**: Consumer vs. B2B vs. Developer
- **Complexity**: Simple MVP vs. Full-featured
- **Industry**: Tech, Finance, Creative, E-commerce, etc.

The generator selects the most appropriate template and customizes:
- Layout structure
- Component priorities
- CTA messaging
- Visual hierarchy
- Content density
