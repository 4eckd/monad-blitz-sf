# GONADS.io Brand Preview - Deployment Guide

## Quick Deploy to Vercel

### Prerequisites
- Vercel account
- Vercel CLI installed: `npm i -g vercel`
- Domain access to configure subdomain

### Deployment Steps

#### 1. Navigate to Preview Directory
```bash
cd brands/gonads-io/preview
```

#### 2. Install Dependencies
```bash
npm install
```

#### 3. Test Locally
```bash
npm run dev
# Open http://localhost:3001
```

#### 4. Deploy to Vercel
```bash
# First deployment
vercel

# Production deployment
vercel --prod
```

#### 5. Configure Custom Domain
1. Go to Vercel Dashboard → Your Project → Settings → Domains
2. Add domain: `gonads.preview.machups.com`
3. Configure DNS:
   - Type: CNAME
   - Name: `gonads.preview` (or subdomain)
   - Value: `cname.vercel-dns.com`
   - TTL: 3600

#### 6. Verify Deployment
```bash
curl https://gonads.preview.machups.com
```

## Alternative: Manual Subdomain Configuration

### Using Cloudflare DNS

1. **Add DNS Record:**
   - Log into Cloudflare dashboard
   - Select `machups.com` zone
   - DNS → Add record:
     - Type: `CNAME`
     - Name: `gonads.preview`
     - Target: `cname.vercel-dns.com`
     - Proxy status: Proxied (orange cloud)
     - TTL: Auto

2. **Verify DNS Propagation:**
```bash
dig gonads.preview.machups.com
nslookup gonads.preview.machups.com
```

3. **Test Deployment:**
```bash
curl -I https://gonads.preview.machups.com
```

## Automated Deployment with GitHub Actions

### Setup Workflow

Create `.github/workflows/deploy-gonads-preview.yml`:

```yaml
name: Deploy Gonads Preview

on:
  push:
    branches:
      - main
    paths:
      - 'brands/gonads-io/**'
  workflow_dispatch:

jobs:
  deploy:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v4

      - name: Setup Node.js
        uses: actions/setup-node@v4
        with:
          node-version: '18'

      - name: Install Dependencies
        working-directory: ./brands/gonads-io/preview
        run: npm install

      - name: Build
        working-directory: ./brands/gonads-io/preview
        run: npm run build

      - name: Deploy to Vercel
        working-directory: ./brands/gonads-io/preview
        env:
          VERCEL_TOKEN: ${{ secrets.VERCEL_TOKEN }}
          VERCEL_ORG_ID: ${{ secrets.VERCEL_ORG_ID }}
          VERCEL_PROJECT_ID: ${{ secrets.VERCEL_PROJECT_ID }}
        run: |
          npx vercel --token $VERCEL_TOKEN --prod --yes
```

### Required Secrets

Add to GitHub Repository Settings → Secrets:
- `VERCEL_TOKEN`: Your Vercel API token
- `VERCEL_ORG_ID`: Your Vercel organization ID
- `VERCEL_PROJECT_ID`: Your Vercel project ID

## Troubleshooting

### Build Fails

```bash
# Clear cache and rebuild
cd brands/gonads-io/preview
rm -rf .next node_modules
npm install
npm run build
```

### Domain Not Resolving

```bash
# Check DNS propagation
dig gonads.preview.machups.com

# Check SSL certificate
curl -vI https://gonads.preview.machups.com
```

### Components Not Loading

```bash
# Verify component paths
ls -la ../components/

# Check tsconfig paths
cat tsconfig.json
```
