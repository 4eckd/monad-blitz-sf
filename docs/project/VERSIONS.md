# Version History

## Versioning Strategy

MACHUPS follows [Semantic Versioning](https://semver.org/): `MAJOR.MINOR.PATCH`

- **MAJOR**: Breaking changes, major feature overhauls
- **MINOR**: New features, backward-compatible
- **PATCH**: Bug fixes, minor improvements
